# Vivienda Ideal Frontend

## Description

Project to create a frontend for the vivienda-ideal API.

Project compose of 2 repositories:

-   [Frontend](https://github.com/Maur025/front-vivienda-ideal-project-db).
-   [Backend](https://github.com/Maur025/back-vivienda-ideal-project-db).

## Project setup

Project require pnpm package manager. If not installed, please install it first.
Cant install pnpm, please follow the instructions on their official website: https://pnpm.io/es/installation

To install all dependencies, run:

```bash
$ pnpm install
```

## Compile and run the project

Run the development server:

```bash
# development
pnpm dev
```

Open [http://localhost:6977](http://localhost:6977) with your browser to see the result.

You can review .env file and update it with your API details. In case not exists, you can create a `.env` file with following content:

```text
BACKEND_HOST=http://localhost
BACKEND_PORT=3000
```
